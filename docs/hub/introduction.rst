Introduction
============

This section explains the purpose of the vantage6 hub and how it works.

What is a vantage6 hub?
-----------------------

A vantage6 hub is the central component of the vantage6 infrastructure. It is
responsible for the coordination of the vantage6 network. It contains the following
components:

- **HQ**: The headquarters of the vantage6 network.
- **Authentication service**: The authentication service for the vantage6 network.
- **Algorithm store**: The store for the algorithms that are used in the vantage6 network.
- **User interface**: The user interface for the vantage6 network.
